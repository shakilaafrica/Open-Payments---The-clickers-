KhuselaKids

KhuselaKids is a system that allows parents to safely transport and monitor their kids safety when transporting their kids on minibus taxis to school. The current system is able to do transfers to wallet using Open Payments. 
- 

## Features

- User registration and profile management
- Open Payments remittance workflow
- Wallet discovery and validation 


## Prequisites

Node.js 20+
npm
An Open Payments compatible wallet
Turso (or SQLite database)

## Installation

npm install

## Run

```bash
npm run dev
```

## Technologies Used

Runtime: Node.js
Language: TypeScript
Framework: Express
Database: SQLite / Turso
ORM: Drizzle ORM
Authentication: JWT
Password Hashing: bcryptjs
Open Payments: @interledger/open-payments
